### 起步：了解老虎机的基本原理

在开始编码之前，我们首先需要了解老虎机的基本工作原理。简单来说，老虎机的工作流程主要包括三个部分：

1. **随机数生成**：这是老虎机的核心机制。玩家按下按钮后，系统会随机生成一组数字，对应到老虎机的图案上。
2. **图案匹配**：根据随机生成的数字，系统会匹配相应的图案。如果这些图案符合特定的组合规则（例如三个相同的图案），则玩家获胜。
3. **奖品计算**：根据不同的组合规则，系统会计算出玩家应该获得的奖励金额。

#### 第一步：定义基本变量和函数

#### 第四步：整合所有功能

最后，我们需要将所有功能整合在一起，形成一个完整的老虎机游戏。下面是一个完整的示例代码：

```python
import random

# 定义图案
symbols = ["🍒", "🍊", "🍋", "🍉", "🍇"]
# 定义奖金表
prize_table = {
    ("🍒", "🍒", "🍒"): 50,
    ("🍊", "🍊", "🍊"): 30,
    ("🍋", "🍋", "🍋"): 20,
    ("🍉", "🍉", "🍉"): 10,
    ("🍇", "🍇", "🍇"): 5
}

def spin():
    return [random.choice(symbols) for _ in range(3)]

def check_win(result):
    if tuple(result) in prize_table:
        return prize_table[tuple(result)]
    else:
        return 0

def main():
    while True:
        input("按回车键开始游戏：")
        result = spin()
        print("结果：", *result)
        win_amount = check_win(result)
        if win_amount > 0:
            print(f"恭喜！你赢了 {win_amount} 元！")
        else:
            print("很遗憾，这次没有中奖。")

if __name__ == "__main__":
    main()
```
